# SkillSpring - Online Learning & Certification Platform

SkillSpring is a responsive web-based learning platform designed to provide users with access to online courses and certifications through a clean and user-friendly interface.

## Features
- Home page with course listings
- Login and Signup pages
- Detailed course description pages
- Responsive design using Bootstrap

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap

## How to Run
1. Download or clone the repository
2. Open secong.html in a browser
